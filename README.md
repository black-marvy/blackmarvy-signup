# blackmarvy-signup
Responsive Sign-Up &amp; Sign-In Form built with HTML, CSS, and JavaScript. Users can create an account and log in using localStorage. Fully mobile-friendly with clean, modern design.
This project demonstrates a basic user authentication system using frontend technologies. The form validates password confirmation, stores user credentials in localStorage, and is fully responsive across desktop, tablet, and mobile screens. Perfect for learning web development fundamentals and UI/UX design.
